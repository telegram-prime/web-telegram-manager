# WBTG – Web Telegram Manager

**Desktop Manager for Multiple Telegram Web Sessions**

Powerful desktop application for managing, organizing, monitoring, and operating multiple Telegram Web accounts from a single unified interface.

WBTG turns chaotic browser tabs into a clean, persistent, and professional workspace for multi-account Telegram Web operations.

---
### 🌟 Why WBTG?

Instead of opening dozens of browser profiles manually, WBTG creates and manages **isolated Telegram Web sessions** using web-profiles — each with its own persistent storage, proxy settings, fingerprint configuration, and operational state.

Perfect for multi-account operators who need reliability, control, and efficiency.

---
### ⚙️ Core Capabilities

| Category                    | Features |
|----------------------------|----------|
| **Multi-Account Management** | Load, organize and restore dozens of Telegram Web accounts in one app |
| **Isolated Profiles**        | Each account runs in its own persistent profile directory (data survives restarts) |
| **2FA-Free Session Import**  | Import existing Telegram sessions into the app without re-entering Telegram 2FA during transfer |
| **Portable Workspace**       | Move the whole application folder with profiles and config to another location without remapping account paths |
| **Recipient Progress Tracking** | Separate sent, pending, and failed recipients for interrupted runs, audits, and controlled retries |
| **Flexible Dialog Opening**  | Multiple recipient dialog opening modes for better compatibility with different Telegram Web flows |
| **Account Monitoring**       | Real-time tracking of unread messages, frozen accounts, and logged-out states |
| **Smart Sender**             | Send text, media, and bot commands with delays, limits, and queue tracking |
| **Media Sending**            | Full image sending workflow with captions, folder support, and sequential/random modes |
| **Scheduling**               | Built-in task scheduler with rescheduling and cancellation options |
| **Proxy Management**         | Bulk proxy assignment (multi-line), per-account proxies, sequential/random distribution |
| **Logging & Reporting**      | Detailed real-time logs, exportable summaries, per-account delivery reports |
| **Environment Control**      | Per-account zoom, fingerprint settings, language/timezone simulation |
| **Session Persistence**      | Full session restoration on startup, portable profile paths |
| **Responsive Support**       | Fast operational support channels for setup issues, license questions, and runtime assistance |
| **Auto Updates**             | Built-in update checks with direct download links for the latest program version |

---
### 🚀 Key Advantages

- **True isolation** — Every account has its own dedicated storage folder
- **Operational Sender** — Advanced sending workflow with progress tracking and interruption recovery
- **Persistent Workspace** — Accounts, settings, and logs are saved between sessions
- **Professional Monitoring** — Instantly see which accounts are active, frozen, or require re-login
- **Flexible Scheduling** — Run campaigns at specific times without constant supervision
- **Built for Windows** — Clean Qt-based desktop application with modern interface
